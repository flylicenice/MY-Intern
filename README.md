# MyIntern Tracking System

## Overview

MyIntern Tracking System is a web-based internship management platform designed to streamline the internship application and monitoring process for students, lecturers, companies, and administrators. The system provides a centralized platform where students can search and apply for internships, track application progress, manage documents, and communicate with employers, while lecturers and companies can efficiently monitor and manage internship-related activities.

The platform aims to reduce administrative burdens, improve transparency, and provide a structured internship experience that better prepares students for future employment.

---

## Problem Statement

The traditional internship application process presents several challenges:

* **Manual Progress Tracking**

  * Students manually update application statuses, resulting in missed interviews, meetings, or deadlines.

* **Application "Black Hole"**

  * Students often receive no updates after applying, creating uncertainty and delaying decision-making.

* **Limited Lecturer Monitoring**

  * Lecturers have difficulty tracking student application progress, making it challenging to provide timely guidance.

---
